NOTES
────────────────────────────────────────────────────────────────────────
• Several photos include identifiable customers (including minors in the
  Parents & Kids and Recent Activity slots). Confirm you have model-release
  permission before publishing publicly.
• All photos lazy-load and use object-fit:cover, so any aspect ratio works.
• To swap a slot, drop a new image with the same filename — no HTML edit
  needed. To add or remove gallery tiles, edit 5th-quarter-cards-prototype.html
  and look for the section labeled "grid-3card-01" with class "gallery-tile".
